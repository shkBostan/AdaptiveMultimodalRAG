# AdaptiveMultimodalRAG

A multimodal Retrieval-Augmented Generation (RAG) system that adaptively handles text, images, and audio inputs.

## Project Structure

This project combines Python and Java implementations with a modern web UI for multimodal RAG capabilities.

## Features

- Multimodal embedding generation (text, images, audio)
- Adaptive retrieval engine
- RAG-based generation
- Web-based UI for interaction
- RESTful API for integration
